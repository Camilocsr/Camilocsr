## 👤 Sobre mí

Basado en Bogotá, Colombia. Trabajo con startups y empresas construyendo sistemas backend, integrando IA y desplegando software en producción. Disponible para proyectos freelance internacionales y consultoría.

## 🔨 Qué construyo

### 🏗️ Ingeniería Backend & Arquitectura Cloud

- Diseñé y desplegué arquitecturas de microservicios distribuidos usando gRPC, Protocol Buffers, GraphQL y REST — con comunicación entre servicios, contratos de esquema y pipelines de despliegue independientes.
- Construí backends SaaS multi-tenant con aislamiento completo de tenants, cifrado de datos AES-256 y rotación de claves AWS KMS para seguridad de datos a nivel de cumplimiento.
- Aprovisioné y administré infraestructura AWS en EC2, S3, CloudFront CDN, Lambda, ALB, IAM, CloudWatch y VPC — incluyendo scripts de aprovisionamiento automatizado y patrones de infraestructura como código.
- Configuré API gateways con Traefik y Nginx para limitación de tasa, enrutamiento de proxy inverso, terminación SSL y balanceo de carga en servicios contenerizados.
- Construí pipelines CI/CD con GitHub Actions y Docker Compose — pruebas automatizadas, builds de imágenes y despliegues sin tiempo de inactividad en Vercel, Railway y AWS.
- Implementé orquestación de flujos de trabajo con Temporal.io usando el patrón Saga para gestión de transacciones distribuidas, lógica de reintentos y flujos de compensación de respaldo.
- Gestioné flujos de trabajo Git a escala: estrategias de ramificación, revisión de código basada en PRs, etiquetado de versiones y pipelines de hotfix en equipos de múltiples desarrolladores.

### 🔐 Ingeniería de Seguridad

- Implementé sistemas de autenticación y autorización usando JWT, OAuth 2.0 y RBAC — con flujos de refresco de tokens, control de acceso basado en alcances y gestión de sesiones.
- Apliqué cifrado en reposo y en tránsito: AES-256 para datos sensibles, TLS/SSL para toda la comunicación entre servicios, AWS KMS para gestión del ciclo de vida de claves y URLs prefirmadas para acceso seguro a objetos S3.
- Endurecí superficies de API con aislamiento de puertos, políticas CORS, limitación de tasa, saneamiento de entradas y gestión de secretos mediante bóvedas de entorno y AWS Secrets Manager.
- Realicé auditorías de seguridad incluyendo escaneo de vulnerabilidades, auditoría de dependencias y remediación de CVEs en stacks de servicios Node.js y Python.

### 🤖 IA, Sistemas de Voz & IA Conversacional

- Construí integraciones con la plataforma de Meta — WhatsApp Business API, Instagram Graph API y Messenger Platform — para flujos conversacionales omnicanal, manejo de eventos webhook y enrutamiento de mensajes a escala.
- Diseñé sistemas de IA conversacional incluyendo chatbots con NLP, pipelines de clasificación de intenciones y resumen de llamadas con IA con salida JSON estructurada para ingesta en CRM.
- Integré ElevenLabs API para síntesis de texto a voz de alta fidelidad y construí pipelines de clonación de voz personalizados en Python para experiencias de voz de marca.
- Trabajé con Coqui TTS (código abierto) para desplegar motores TTS auto-alojados — incluyendo ajuste fino, servicio de modelos y pipelines de salida de audio en streaming.
- Implementé pipelines de voz a texto usando OpenAI Whisper para transcripción por lotes y AWS Transcribe para transcripción en streaming en tiempo real con diarización de hablantes.
- Integré Telnyx VoIP para control de llamadas programable — troncales SIP, grabación de llamadas, transcripción en tiempo real, manejo de medios y orquestación de flujos de llamadas basada en webhooks.
- Construí sistemas de agentes de IA con LangChain — pipelines RAG, agentes con llamadas a herramientas, ingeniería de prompts y generación aumentada por recuperación sobre fuentes de datos estructuradas y no estructuradas.
- Usé OpenAI API para generación de salida estructurada, llamadas a funciones y desarrollo de funcionalidades impulsadas por IA en productos SaaS en producción.

### 📡 Mensajería, Tiempo Real & Analítica

- Construí infraestructura de mensajería omnicanal abarcando WhatsApp, Instagram, Messenger y VoIP — con enrutamiento unificado de mensajes, seguimiento de entrega y gestión de sesiones multicanal.
- Implementé arquitecturas orientadas a eventos con WebSockets para comunicación en tiempo real — dashboards en vivo, notificaciones push y flujos de datos bidireccionales entre clientes y servicios backend.
- Diseñé pipelines de procesamiento de audio: ingesta de grabaciones de llamadas, orquestación de transcripción, entrega de audio en streaming y distribución S3 + CloudFront CDN para reproducción de baja latencia.
- Construí dashboards de analítica en tiempo real con seguimiento de KPIs, visualización de datos e informes BI — exponiendo métricas operativas a stakeholders de negocio con tasas de actualización de sub-segundo.
- Implementé sistemas de analítica de campañas con seguimiento de conversiones, frameworks de pruebas A/B e integración de Meta Pixel para atribución y análisis de embudo.
- Configuré stacks de monitoreo y observabilidad usando CloudWatch, pipelines de agregación de logs, reglas de alertas y dashboards de seguimiento de rendimiento para la salud de servicios en producción.

### 🚀 Ingeniería de Producto & Plataforma

- Construí plataformas de e-commerce con facturación por suscripción vía MercadoPago, gestión de inventario y personalización de tienda — manejando webhooks de pago, ciclo de vida de suscripciones y gestión de catálogo.
- Desarrollé sistemas CRM con dashboards de analítica, tableros de pipeline estilo kanban y métricas en tiempo real — dando a los equipos de ventas y operaciones visibilidad completa de sus flujos de trabajo.
- Implementé pipelines de automatización de marketing usando Meta Ads API — gestión de campañas, pruebas A/B, segmentación de audiencias y seguimiento de eventos Meta Pixel para optimización de rendimiento.
- Construí sistemas de gestión de leads con ingesta de Excel/CSV, secuencias de alcance automatizado y dashboards de reportes — procesando listas masivas de contactos en flujos de trabajo CRM estructurados.
- Contribuí a proyectos de código abierto incluyendo Coqui TTS — enviando correcciones, mejoras e integraciones que benefician a la comunidad de desarrolladores en general.
- Entregué productos SaaS completos desde la arquitectura hasta el despliegue — siendo dueño del ciclo de vida completo desde el diseño del esquema de base de datos hasta el monitoreo en producción.

### 🎮 Desarrollo de Videojuegos & Sistemas Interactivos

Construí aplicaciones Unity 3D para experiencias educativas de VR/AR — incluyendo sistemas de avatares interactivos y aplicaciones de aprendizaje de idiomas con bucles de retroalimentación de IA en tiempo real. Desplegado en Android y WebGL.
