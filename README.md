## 🔨 What I Build

### 🏗️ Backend Engineering & Cloud Architecture

- Designed and deployed distributed microservices architectures using gRPC, Protocol Buffers, GraphQL, and REST — with service-to-service communication, schema contracts, and independent deployment pipelines.
- Built multi-tenant SaaS backends with full tenant isolation, AES-256 data encryption, and AWS KMS key rotation for compliance-grade data security.
- Provisioned and managed AWS infrastructure across EC2, S3, CloudFront CDN, Lambda, ALB, IAM, CloudWatch, and VPC — including automated provisioning scripts and infrastructure-as-code patterns.
- Configured API gateways with Traefik and Nginx for rate limiting, reverse proxy routing, SSL termination, and load balancing across containerized services.
- Built CI/CD pipelines with GitHub Actions and Docker Compose — automated testing, image builds, and zero-downtime deployments to Vercel, Railway, and AWS.
- Implemented workflow orchestration with Temporal.io using the Saga pattern for distributed transaction management, retry logic, and fallback compensation flows.
- Managed Git workflows at scale: branching strategies, PR-based code review, release tagging, and hotfix pipelines across multi-developer teams.

### 🔐 Security Engineering

- Implemented authentication and authorization systems using JWT, OAuth 2.0, and RBAC — with token refresh flows, scope-based access control, and session management.
- Applied encryption at rest and in transit: AES-256 for sensitive data, TLS/SSL for all service communication, AWS KMS for key lifecycle management, and presigned URLs for secure S3 object access.
- Hardened API surfaces with port isolation, CORS policies, rate limiting, input sanitization, and secrets management via environment vaults and AWS Secrets Manager.
- Conducted security audits including vulnerability scanning, dependency auditing, and remediation of CVEs across Node.js and Python service stacks.

### 🤖 AI, Voice Systems & Conversational AI

- Built integrations with Meta's platform — WhatsApp Business API, Instagram Graph API, and Messenger Platform — for omnichannel conversational flows, webhook event handling, and message routing at scale.
- Engineered conversational AI systems including NLP-powered chatbots, intent classification pipelines, and AI call summarization with structured JSON output for downstream CRM ingestion.
- Integrated ElevenLabs API for high-fidelity text-to-speech synthesis and built custom voice cloning pipelines in Python for branded voice experiences.
- Worked with Coqui TTS (open source) to deploy self-hosted TTS engines — including fine-tuning, model serving, and streaming audio output pipelines.
- Implemented speech-to-text pipelines using OpenAI Whisper for batch transcription and AWS Transcribe for real-time streaming transcription with speaker diarization.
- Integrated Telnyx VoIP for programmable call control — SIP trunking, call recording, real-time transcription, media handling, and webhook-driven call flow orchestration.
- Built AI agent systems with LangChain — RAG pipelines, tool-calling agents, prompt engineering, and retrieval-augmented generation over structured and unstructured data sources.
- Used OpenAI API for structured output generation, function calling, and AI-powered feature development across production SaaS products.

### 📡 Messaging, Real-Time & Analytics

- Built omnichannel messaging infrastructure spanning WhatsApp, Instagram, Messenger, and VoIP — with unified message routing, delivery tracking, and multi-channel session management.
- Implemented event-driven architectures with WebSockets for real-time communication — live dashboards, push notifications, and bidirectional data streams between clients and backend services.
- Engineered audio processing pipelines: call recording ingestion, transcription orchestration, streaming audio delivery, and S3 + CloudFront CDN distribution for low-latency playback.
- Built real-time analytics dashboards with KPI tracking, data visualization, and BI reporting — surfacing operational metrics to business stakeholders with sub-second refresh rates.
- Implemented campaign analytics systems with conversion tracking, A/B testing frameworks, and Meta Pixel integration for attribution and funnel analysis.
- Set up monitoring and observability stacks using CloudWatch, log aggregation pipelines, alerting rules, and performance tracking dashboards for production service health.

### 🚀 Product & Platform Engineering

- Built e-commerce platforms with subscription billing via MercadoPago, inventory management, and storefront personalization — handling payment webhooks, subscription lifecycle, and catalog management.
- Developed CRM systems with analytics dashboards, kanban-style pipeline boards, and real-time metrics — giving sales and operations teams full visibility into their workflows.
- Implemented marketing automation pipelines using Meta Ads API — campaign management, A/B testing, audience segmentation, and Meta Pixel event tracking for performance optimization.
- Built lead management systems with Excel/CSV ingestion, automated outreach sequencing, and reporting dashboards — processing bulk contact lists into structured CRM workflows.
- Contributed to open source projects including Coqui TTS — submitting fixes, improvements, and integrations that benefit the broader developer community.
- Shipped complete SaaS products from architecture through deployment — owning the full lifecycle from database schema design to production monitoring.

### 🎮 Game Dev & Interactive Systems

Built Unity 3D applications for VR/AR educational experiences — including interactive avatar systems and language learning apps with real-time AI feedback loops. Shipped to Android and WebGL targets.
